# aqar-scraper
python script to scrape [Aqar](https://sa.aqar.fm/)


## get started 

### Clone the repo :
```
git clone https://github.com/NafieAlhilaly/aqar-scraper.git
cd aqar-scraper
```

### Install dependencies
using pip :
```
pip install -r requirements.txt
```

using poetry :
```
poetry install
```

### run the script 
```
python app/app.py
```
